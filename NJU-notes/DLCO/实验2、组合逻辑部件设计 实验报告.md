# 实验目的
1. 掌握译码器、编码器、数码管等组合部件的设计方法。
2. 掌握 1 位加法器、4 位串行加法器的设计方法。
3. 掌握补码减法运算方法，实现补码减法功能。
4. 掌握汉明码编码和纠错的设计方法。
5. 掌握桶形移位器的设计方法。
# 实验环境
Logisim 2.16
# 2.1 编写3-8译码器
## 整体方案设计
### 顶层模块设计
根据 3-8 译码器芯片 74X138 的电路原理图，设计一个由反相逻辑门电路构成的 3-8 译码器，并对电路进行仿真测试，以验证电路的功能。本电路采用负逻辑输出。
### 输入输出引脚
| 引脚              | 功能       |
| --------------- | -------- |
| G1              | 高电平有效使能端 |
| G2A_L/G2B_L     | 低电平有效使能端 |
| A               | 二进制低位输入  |
| B               | 二进制中位输入  |
| C               | 二进制高位输入  |
| Yi_L(i=0,1,…,7) | 低电平有效输出端 |

# 2.5 编写3-8译码器
## 整体方案设计
### 顶层模块设计
设计一个 8 位桶形移位器，支持算术左移、逻辑左移、算术右移和逻辑右移四种操作。
### 输入输出引脚

**表：8位桶形移位器引脚功能表**

| 引脚           | 功能       |
| ------------ | -------- |
| datain[7:0]  | 输入数据位    |
| shamt[2:0]   | 移位位数     |
| L/R          | 左右移选择    |
| A/L          | 算术/逻辑移选择 |
| dataout[7:0] | 输出数据位    |

## 错误现象及分析
以为L/R连的是四路选择器的使能端，然后把shamt对应位连到选择端，出现位数不匹配的问题，然后重新梳理逻辑，发现应该将L/R与shamt对应位合成一个二位数输入选择端，不需要使能端。
