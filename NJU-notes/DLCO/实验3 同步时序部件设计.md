# 3.1 计数器实验
## 整体方案设计
### 顶层模块设计
根据表 3.1 给出的功能表和图 3.1 所示电路原理图构建 4 位二进制同步计数器电路。
### 输入输出引脚

| 引脚          | 功能             |
| ----------- | -------------- |
| CLK         | 时钟信号           |
| CLR         | 同步清零端，强制让计数器归零 |
| LD          | 同步并行置数端        |
| ENT和ENP     | 计数使能端          |
| D0、D1、D2、D3 | 置数             |
| Q0、Q1、Q2、Q3 | 表示计数器的当前状态     |

# 3.3 4位无符号乘法器
## 整体方案设计
### 顶层模块设计
实现两个四位二进制无符号数相乘的功能，并通过数码管将其转换成十六进制显示出来。
### 输入输出引脚
- XY：乘数和被乘数
- CLK：时钟信号
- Z：计算结果
- RST：重置电路，并加载新的乘数和被乘数

## 错误现象及分析
stop变为1之后继续变更CLK，发现z还在不断变换。思考之后觉得应该要让stop为1时CLK不再对电路施加影响，由此使用与门，并将stop取反作为4位同步计数器的使能端，从而在stop变为1后固定stop的状态。
# 3.4 寄存器堆
## 整体方案设计
### 顶层模块设计
构建含有32个32位寄存器的寄存器堆Regfile的读写电路，包含两个读数据端口和一个写数据端口，并封装成子电路。
### 输入输出引脚

| 引脚  | 功能            |
| --- | ------------- |
| CLK | 时钟信号          |
| WE  | 写使能信号         |
| RW  | 写口的寄存器编号      |
| RA  | 读口1的寄存器编号     |
| RB  | 读口2的寄存器编号     |
| Din | 输入数据          |
| RDA | RA对应的寄存器输出的数据 |
| RDB | RB对应的寄存器输出的数据 |
