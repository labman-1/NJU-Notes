# 实验目的
1. 掌握 Logisim 软件的使用方法和实验步骤。
2. 掌握 CMOS 晶体管的特性以及逻辑门电路的构建法。
3. 掌握使用不同组件实现多路选择器的设计方法。
4. 掌握电路外观设计方法。
5. 掌握子电路级联设计方法。
# 实验环境
Logisim：https://github.com/Logisim-Ita/Logisim

# 实验3：2选1多路选择器(1)
## 整体方案设计

### 顶层模块设计
利用基本逻辑门和 CMOS 晶体管实现多路选择器，并进行静态冒险检测。根据 2 选 1 多路选择器（2 路选择器）的逻辑表达式$ $Y=D0\cdot \overline S + D1\cdot S$，使用两个 2

输入与门、1 个 2 输入或门、1 个非门、3 个输入端和 1 个输出端实现两级与-或逻辑电路。

### 输入输出引脚

| --- | ---- |
| D0  | 输入0  |
| D1  | 输入1  |
| S   | 选择信号 |
| Y   | 输出   |
