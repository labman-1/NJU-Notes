# 实验目的
1. 掌握 Logisim 软件的使用方法和实验步骤。
2. 掌握 CMOS 晶体管的特性以及逻辑门电路的构建法。
3. 掌握使用不同组件实现多路选择器的设计方法。
4. 掌握电路外观设计方法。
5. 掌握子电路级联设计方法。
# 实验环境
Logisim：https://github.com/Logisim-Ita/Logisim
# 实验1：3输入多数表决器
## 整体方案设计
### 顶层模块设计
功能：当输入X、Y、Z中至少有两个为1时，输出F为1，否则输出F为0.
表达式：$F=XY+XZ+YZ$
### 输入输出引脚
| 引脚  | 功能  |
| --- | --- |
| X   | 输入  |
| Y   | 输入  |
| Z   | 输入  |
| F   | 输出  |
