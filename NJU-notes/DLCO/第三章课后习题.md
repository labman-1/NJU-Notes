(3)存在竞争冒险。
观察卡诺图可知，$AB$ 项与 $\bar{B}\bar{D}$ 项在 $B$ 变量上互补且相邻。当 $A=1, D=0, C=0$ 时，函数简化为 $F = B + \bar{B}$。由于实际电路中非门存在传输延迟，当 $B$ 发生电平跳变时，两项可能同时为 0，从而在输出端 $F$ 产生向 0 跳变的负毛刺

#### 4. 如何消除？（虽然题目没问，但建议写在笔记里）
加入**冗余项（Consensus Term）**。
在卡诺图上把那两个圈“相切”的地方再补一个圈。对于 $AB$ 和 $\bar{B}\bar{D}$，补一个 $A\bar{D}$。
修正后的式子：$F = AB + AC + \bar{B}\bar{D} + \mathbf{A\bar{D}}$。
