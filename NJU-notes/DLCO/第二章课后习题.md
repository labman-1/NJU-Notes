(T2)证明 $X + 1 = 1$

|  $X$  | 左边表达式：$X + 1$ | 右边表达式：$1$ | 左右是否相等？ |     |     |
| :---: | :-----------: | :-------: | :-----: | --- | --- |
| **0** |  $0 + 1 = 1$  |    $1$    |   相等    |     |     |
| **1** |  $1 + 1 = 1$  |    $1$    |   相等    |     |     |
得证
（T2D）证明$X·0=0$

|  $X$  | 左边表达式：$X · 0$ | 右边表达式：$0$ | 左右是否相等？ |     |     |
| :---: | :-----------: | :-------: | :-----: | --- | --- |
| **0** |  $0 ·0 = 0$   |    $0$    |   相等    |     |     |
| **1** |  $1· 0 = 0$   |    $0$    |   相等    |     |     |
得证
（T3）证明 同一律 $X + X = X$
**当 $X = 0$ 时**：
    *   左边 = $0 + 0 = 0$
    *   右边 = $0$
    *   左边 = 右边
*   **当 $X = 1$ 时**：
    *   左边 = $1 + 1 = 1$ 
    *   右边 = $1$
    *   左边 = 右边
*   **结论**：穷举了 $X$ 的所有取值，等式均成立，故 $X + X = X$ 得证。
（T3D）证明$X·X=X$

|  $X$  | 左边表达式：$X · X$ | 右边表达式：$X$ | 左右是否相等？ |
| :---: | :-----------: | :-------: | :-----: |
| **0** |  $0 ·0 = 0$   |    $0$    |   相等    |
| **1** |  $1· 1 = 1$   |    $1$    |   相等    |
得证
（T5）证明 互补律$X \cdot \overline{X} = 0$
*   **当 $X = 0$ 时**，$\overline{X} = 1$：
    *   左边 = $0 \cdot 1 = 0$
    *   右边 = $0$
    *   左边 = 右边
*   **当 $X = 1$ 时**，$\overline{X} = 0$：
    *   左边 = $1 \cdot 0 = 0$
    *   右边 = $0$
    *   左边 = 右边
*   **结论**：穷举了 $X$ 的所有取值，等式均成立，故 $X \cdot \overline{X} = 0$ 得证。
