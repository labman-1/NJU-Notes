# 12.
首先，根据电路图写出状态转移方程。
观察图 4.25：这是一个右移一位寄存器，数据流向是：$X \to Q_3 \to Q_2 \to Q_1 \to Q_0$。

1.  **反馈逻辑**：题目给定 $X = Q_3 \oplus Q_2$。
2.  **转移方程**（假设当前时刻为 $n$，下一时刻为 $n+1$）：
    *   $Q_3^{n+1} = X = Q_3^n \oplus Q_2^n$
    *   $Q_2^{n+1} = Q_3^n$
    *   $Q_1^{n+1} = Q_2^n$
    *   $Q_0^{n+1} = Q_1^n$

---
然后，可以从一个非零状态1000开始推导。
状态追踪表 ($Q_3 Q_2 Q_1 Q_0$)：

| 步骤  | 当前状态     | $X = Q_3 \oplus Q_2$ | 下一状态             |
| :-- | :------- | :------------------- | :--------------- |
| 1   | **1000** | $1 \oplus 0 = 1$     | **1100**         |
| 2   | **1100** | $1 \oplus 1 = 0$     | **0110**         |
| 3   | **0110** | $0 \oplus 1 = 1$     | **1011**         |
| 4   | **1011** | $1 \oplus 0 = 1$     | **1101**         |
| 5   | **1101** | $1 \oplus 1 = 0$     | **0110** (回到步骤2) |

分析状态转移过程：
*   该电路进入了一个长度为 **3** 的死循环：`0110 -> 1011 -> 1101 -> 0110`。
*   初始状态 `1000` 经过两步后进入了这个循环。

再检查特殊状态：
*   0000：$X = 0 \oplus 0 = 0$。下一状态还是 `0000`。
    *   *结论*：全 0 状态是一个“死锁”状态。
*   0001：$X=0 \to$ 下一个状态 `0000`。
    *   *结论*：有些状态会汇聚到全 0 死锁中。

最后再来总结其特点：
1.  非最大长度序列：
    *   一个 4 位的 LFSR 理论上最长循环可以是 $2^4 - 1 = 15$。但本题由于反馈多项式选择不当（相当于 $x^4 + x + x^2$），它的主循环长度只有 3。
2.  存在死区：
    *   全 0 状态是无法自启动的。一旦进入 0000，就会永远保持 0000。
    *   部分状态（如 `0001`）会退化到全 0 状态。
3.  伪随机性差：
    *   由于循环周期极短，它产生的序列（例如观察 $Q_0$ 的输出）重复性极高，不具备良好的随机分布特性。
4.  结构简单：
    *   仅需一个异或门即可实现，体现了移位寄存器作为计数器的低硬件开销优势。
