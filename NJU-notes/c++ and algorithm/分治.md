# 定义
**分治算法（Divide and Conquer）**：即「分而治之」，把一个复杂问题拆分成多个相同或相似的子问题，递归分解，直到子问题足够简单可以直接解决，最后将子问题的解合并得到原问题的解。
简而言之，分治算法就是：**把大问题不断拆小，直到可以直接求解，再合并结果**。

# 例题1 [P1429 平面最近点对（加强版） - 洛谷](https://www.luogu.com.cn/problem/P1429)
这是分治法中“合”的精髓：**“既然我已经保底有一个最优解 $d$ 了，任何在某一个维度上差距就已经大于 $d$ 的候选人，我连看都不看！”** 因为水平距离 $\Delta x > d$，那真实距离 $\sqrt{\Delta x^2 + \Delta y^2}$ 肯定也 $>d$。这叫**最优性剪枝**。

但是，只限制了 $\Delta x \le d$ 还不够。万一所有的点都密密麻麻地挤在这个宽度为 $2d$ 的长条地带里呢？如果我们在长条里两两比对，时间复杂度依然会退化成 $O(N^2)$。

这就需要请出你问到的**“鸽巢原理”**来救场了！

---

### 🕊️ 核心概念一：鸽巢原理 (Pigeonhole Principle) 及其在平面最近点对中的魔法

**鸽巢原理（抽屉原理）的通俗定义：**
“如果有 5 只鸽子，只有 4 个鸽巢，那么不管你怎么塞，**至少有 1 个鸽巢里挤了 2 只（或以上）的鸽子。**”
反过来说：“如果你要求每个鸽巢最多只能住 1 只鸽子，那么 4 个鸽巢最多只能装 4 只鸽子。”

**魔法应用：如何避免在 $2d$ 宽的长条里发生 $O(N^2)$ 惨案？**

假设我们把长条里的点，按照 $Y$ 坐标从下到上排好序。
现在轮到检查长条里的点 $P$ 了。为了找有没有比 $d$ 更近的点，我们只需要往上看，找那些 $Y$ 坐标在 $[P.y, P.y + d]$ 范围内的点（因为 $Y$ 差距超过 $d$ 的，真实距离肯定也 $>d$）。

**学姐的证明时间：**
我们现在锁定了 $P$ 上方一个宽度为 $2d$（横跨左右半区）、高度为 $d$ 的矩形区域。
这个大矩形，可以切成左右两个 $d \times d$ 的正方形。
*   根据我们在“分治”阶段的保证：**左半区内部的任意两个点，距离至少为 $d$；右半区内部也是。**
*   **思考：** 在一个 $d \times d$ 的正方形里，如果你想塞进一堆点，且要求任意两点之间的距离都 $\ge d$，你能塞几个？
*   **答案：** 最多塞 **4** 个！（只能放在四个角落里。如果放第 5 个，根据鸽巢原理，必然会有两个点挤在同一个被划分成 4 块的小正方形里，它们的距离必然 $< d$）。

**终极结论：**
左右两个正方形，每个最多装 4 个点。加起来最多 8 个点。
也就是说，在按 $Y$ 坐标排好序后，对于长条里的每一个点 $P$，**它上方可能和它构成最近点对的候选人，绝对不会超过 7 个！**

这太震撼了！这意味着我们在合并的时候，内层循环**只需要往后看 7 个点即可 `break`**！这就是为什么合并操作从 $O(N^2)$ 变成了 $O(N)$。加上外层排序，总复杂度完美达到 $O(N \log N)$！
你现在已经掌握了**分治法处理最近点对** 的最核心机密：
1.  按 $X$ 排序，递归劈成两半，求出左右最小距离 $d = \min(d_1, d_2)$。
2.  把横向距离中线不超过 $d$ 的点挑出来。
3.  把这些挑出来的点按 $Y$ 排序。
4.  遍历这些点，每个点只和它后面 $Y$ 距离不超过 $d$ 的点（鸽巢原理保证最多几个）计算真实距离，更新 $d$。
```c++
#include<bits/stdc++.h>
# define MAX 200005
using namespace std;
int n;
double d=0x3f3f3f3f;
struct dot {
	long long x;
	long long y;
	dot() {
		x = 0;
		y = 0;
	}
	dot(long  a, long long b) :x(a), y(b) {}
	~dot(){}
};
vector<dot>dots(MAX);
double distance(const dot& a, const dot& b) {
	double di = sqrt((a.x - b.x) * (a.x - b.x) + (a.y-b.y)*(a.y-b.y));
	return di;
}
double divide(int start, int end) {
	int length =end - start + 1;
	if (length == 2) {
		return distance(dots[start], dots[end]);
	}
	else if (length == 3) {
		double d1 = distance(dots[start], dots[end]);
		double d2 = distance(dots[start], dots[end-1]);
		double d3 = distance(dots[start+1], dots[end]);
		return min(d1, min(d2, d3));
	}
	else if (length > 3) {
		int mid = start + ((length - 1) >> 1);
		double d1 = divide(start, mid);
		double d2 = divide(mid + 1, end);
		double mind = min(d1, d2);
		vector<dot>strip;
		//此处必须有[start,end]，end必须也进入候选人名单
		for (int i = start; i <= end; i++) {
			if (abs(dots[i].x - dots[mid].x) <= mind) {
				strip.push_back(dots[i]);
			}
		}
		sort(strip.begin() , strip.end(), [](dot& a, dot& b) {return a.y < b.y;});
		for (int i = 0;i < strip.size();i++) {
			for (int j = i+1;j < strip.size() && strip[j].y <= (strip[i].y + mind);j++) {
				mind = min(mind, distance(strip[i], strip[j]));
			}
		}
		return mind;
	}
	else {
		return 0x3f3f3f3f;
	}
}
int main() {
	ios::sync_with_stdio(false);cin.tie(0);
	cin >> n;
	for (int i = 0;i < n;i++) {
		int x, y;
		cin >> x >> y;
		dot temp(x, y);
		dots[i] = temp;
	}
	d = distance(dots[0], dots[1]);
	sort(dots.begin(), dots.begin()+n, [](dot& a, dot& b) {return a.x < b.x;});
	d = divide(0,n-1);
	cout << fixed << setprecision(4) << d;
	return 0;
}
```

long long binPow(long long a, long long b, long long p) {
	long long res = 1;
	while (b > 0) {
		if (b & 1) res = res * a % p;
		a = a * a%p;
		b=b >> 1;
	}
	return res;
}
