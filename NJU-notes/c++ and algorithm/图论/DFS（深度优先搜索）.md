- **DFS** 通常用来解 **“能不能走到”**、**“遍历所有可能性（暴力枚举）”** 或者 **“连通块个数”**。
- ####  DFS（深度优先）：不撞南墙不回头

- **性格**：执着、一根筋。
    
- **策略**：
    
    1. 看到一条路，就一直往里走，走到死胡同为止。
        
    2. 到了死胡同，退回到上一个路口（回溯），换一条没走过的路继续往里冲。
        
    3. 重复直到找到钥匙。
        
- **数据结构**：**栈 (Stack)**（通常用**递归**系统栈实现）。
    
- **视觉效果**：像一条贪吃蛇，蜿蜒深入。

###  DFS 代码模板（递归版）

DFS 写起来最简单，不需要显式定义栈，直接利用函数的递归调用。

**场景**：给一个图（邻接表），遍历所有点。

```cpp
// 必须有的：vis 数组，防止转圈圈（死循环）
bool vis[10005]; 
vector<int> adj[10005]; // 邻接表

void dfs(int u) {
    // 1. 标记当前点已访问
    vis[u] = true;
    cout << "访问到了: " << u << endl;

    // 2. 遍历 u 的所有邻居
    for (int v : adj[u]) {
        if (!vis[v]) { // 重点：如果邻居没来过，就冲进去
            dfs(v);    // 递归调用
        }
    }
    // 3. (可选) 回溯位置：有时候需要把 vis[u] 改回 false，但在纯遍历中不需要
}
```
