- 它解决的是 **”依赖关系“** 问题。基本上是**BFS的变体**。
- 一句话概括：先把入度为0的点干掉，然后把它的后续影响消除，再找下一个没人管的点。

**算法流程 (Kahn 算法)：**
1.  **统计入度**：遍历所有边 $u \to v$，统计每个点 $v$ 被多少人依赖（`in_degree[v]++`）。
2.  **入队**：把所有 **入度为 0** 的点（没有任何先修课）扔进 `queue`。
3.  **BFS 循环**：
    *   取出队头 $u$（修这门课）。
    *   遍历 $u$ 的邻居 $v$（$u$ 是 $v$ 的先修课）。
    *   把 $v$ 的入度减 1（`in_degree[v]--`，表示 $u$ 这门先修课搞定了）。
    *   **关键**：如果 $v$ 的入度变成了 0（所有先修课都搞定了），把 $v$ 入队。
```c++
#include<bits/stdc++.h>
using namespace std;
const int N = 105;
int n;
vector<int>adj[N];
int in[N] ;
void topo_sort() {
	queue<int>q;
	for (int i = 1; i <= n;i++) {
		if (in[i] == 0) q.push(i);
	}
	while (!q.empty()) {
		int u = q.front();
		q.pop();
		cout << u << " ";
		for (auto i : adj[u]) {
			in[i]--;
			if (in[i] == 0) q.push(i);
		}
	}
}

int main() {
	ios::sync_with_stdio(false);cin.tie(0);
	cin >> n;
	for (int i = 1;i <= n;i++) {
		int offstring;
		while (cin >> offstring && offstring != 0) {
			in[offstring]++;
			adj[i].push_back(offstring);
		}
	}
	topo_sort();
	return 0;
}
```
