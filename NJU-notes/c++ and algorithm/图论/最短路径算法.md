**场景**：地图导航，从A到B怎么走最快？
## Floyd算法
- **“上帝视角的中转站”
- **一句话原理**：我想从i到j，能不能通过中间点k中转一下，使路径变短？（i->k->j比i->j短吗？）
- **特征：
	- **核心：3个for循环
	- **适用**：点很少（N<=500），求**任意两点**间的最短路径。
	- **记忆点**：k（中转点）在最外层循环。

## Dijkstra算法
- **“贪心的探险家”
- **一句话原理**：永远只挑**当前离起点最近**的那个点探索，永不回头。
- **核心关联：BFS plus版**
	- BFS用`queue`（FIFO）
	- Dijkstra用`priority_queue`（谁距离短谁先走）
- **适用**：点很多（N<=1e5），没有负权边，求**单源**最短路径。

###  核心修改：队列 (Queue) $\to$ 优先队列 (Priority Queue)

Dijkstra 的算法流程和 BFS **几乎一模一样**，只有两个关键改动：

1.  **容器换了**：
    *   BFS 用 `std::queue`（先进先出）。
    *   Dijkstra 用 `std::priority_queue`（**谁距离短，谁先出**）。
    *   *人设：Dijkstra 是个势利眼，永远只挑当前离起点最近的那个点去拓展。*

2.  **松弛操作 (Relaxation)**：
    *   BFS：只要没访问过，就标记 `visited` 并入队。
    *   Dijkstra：我不管你访没访问过，只要**我发现了一条更近的路**（`new_dist < old_dist`），我就更新你的距离，并把你再次入队。

###  Dijkstra 标准模板（抄作业专用）

**前置知识点**：
*   我们需要一个小顶堆（最小值在上面）。
*   C++ 的 `priority_queue` 默认是大顶堆。
*   **变身咒语**：`priority_queue<Type, vector<Type>, greater<Type>>`。
*   我们用 `pair<int, int>` 存 `{距离, 点编号}`，因为 pair 默认先比较第一项（距离），正好符合我们的需求。

```cpp
#include <bits/stdc++.h>
using namespace std;

const int INF = 0x3f3f3f3f; // 无穷大
const int N = 100005;      // 最大点数

// 邻接表：pair<邻居, 边权>
vector<pair<int, int>> adj[N]; 
int dist[N]; // 存储起点到各点的最短距离

void dijkstra(int start) {
    // 1. 初始化
    memset(dist, 0x3f, sizeof(dist)); // 全部设为无穷大
    dist[start] = 0;
    
    // 定义小顶堆：存 {距离, 点编号}
    // 写法有点长，建议多敲几遍形成肌肉记忆
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    
    // 起点入队：距离0，编号start
    pq.push({0, start});
    
    while (!pq.empty()) {
        // 2. 取出当前距离最近的点
        pair<int, int> curr = pq.top();
        pq.pop();
        
        int d = curr.first;  // 当前累计的距离
        int u = curr.second; // 当前点的编号
        
        // 3. 懒惰删除（Lazy Deletion）的关键：
        // 如果从堆里取出的距离 d 比我们要更新的 dist[u] 还要大，
        // 说明这是一个“过期”的旧数据（之前被更优的路径更新过了），直接扔掉。
        if (d > dist[u]) continue;
        
        // 4. 扫描所有邻居（和 BFS 一样）
        for (auto edge : adj[u]) {
            int v = edge.first;  // 邻居编号
            int w = edge.second; // 边权 (u到v的距离)
            
            // 5. 松弛操作：如果经由 u 到 v 更近
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w; // 更新距离
                pq.push({dist[v], v}); // 把更优的 {新距离, v} 扔进堆里
            }
        }
    }
}

int main() {
    int n, m, s; // n个点, m条边, s是起点
    cin >> n >> m >> s;
    for (int i = 0; i < m; i++) {
        int u, v, w;
        cin >> u >> v >> w;
        adj[u].push_back({v, w}); // 有向图
        // adj[v].push_back({u, w}); // 如果是无向图，这行也要写
    }
    
    dijkstra(s);
    
    // 输出到各点的距离
    for(int i=1; i<=n; i++) cout << dist[i] << " ";
    return 0;
}
```
