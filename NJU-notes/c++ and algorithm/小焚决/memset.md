
`memset` 是 C/C++ 中**最快**、**最常用**的内存初始化工具。在 OI/ACM 竞赛中，它几乎是写 DP、图论、搜索题时的**标准起手式**。

---

### 1. 它是干什么的？

`memset` 的作用是：**按字节（Byte）** 填充一段内存区域。

*   **头文件**：`<cstring>` (C++) 或 `<string.h>` (C)
*   **语法**：
    ```cpp
    void *memset(void *ptr, int value, size_t num);
    ```
    *   `ptr`：指向要填充的数组或内存块的指针（比如数组名）。
    *   `value`：你要填进去的值（**注意：虽然是 int，但它只取低 8 位，也就是 1 个字节**）。
    *   `num`：要填充的**字节数**（不是元素个数！）。

---

### 2. 核心原理（这里最容易踩坑！）

**千万记住：`memset` 是按“字节”赋值的，不是按“元素”赋值的。**

假设你有一个 `int` 数组（`int` 占 4 个字节）。

#### ✅ 场景一：初始化为 0（清空）
这是最安全的用法。
```cpp
int a[10];
memset(a, 0, sizeof(a)); 
```
*   **原理**：`int` 的 4 个字节都变成 `00000000`。
*   **结果**：`00000000 00000000 00000000 00000000` (二进制) = **0** (十进制)。
*   **用途**：清空 `vis` 数组、清空统计数组、图的邻接矩阵清空。

#### ✅ 场景二：初始化为 -1
这也是安全的。
```cpp
int a[10];
memset(a, -1, sizeof(a));
```
*   **原理**：-1 在计算机里是补码全 1。`memset` 把每个字节都设为 `11111111`（即 0xFF）。
*   **结果**：`11111111 11111111 11111111 11111111` = **-1** (十进制)。
*   **用途**：初始化 DP 数组表示“未访问”或“无解”，初始化链表空指针（如果是用数组模拟）。

#### ❌ 场景三：初始化为 1（经典错误！）
如果你想把数组全部初始化为 1：
```cpp
int a[10];
memset(a, 1, sizeof(a)); // 错误！结果不是 1
```
*   **原理**：`value` 是 1，二进制是 `00000001`。`memset` 会把 `int` 的 4 个字节**都**变成 `00000001`。
*   **结果**：`00000001 00000001 00000001 00000001`。
*   **算一下**：这是十六进制的 `0x01010101`，十进制约为 **16843009**。
*   **结论**：**除了 0 和 -1，千万别用 `memset` 给 int 数组赋其他具体的整数值（如 1, 2, 99 等）。**

### 3. OIer 的“黑科技”：初始化为无穷大 (Infinity) ♾️

在你做**最短路 (Dijkstra/Floyd)** 或 **DP 求最小值** 时，经常需要把数组初始化为一个“无穷大”的数。

新手通常会用 `for` 循环赋值 `INT_MAX`，或者 `memset` 赋值 `0x7f`。
推荐一个**竞赛圈通用的魔法值：`0x3f`**。

```cpp
const int INF = 0x3f3f3f3f; // 约等于 10^9
int dist[1005];

// 把 dist 数组全部初始化为“无穷大”
memset(dist, 0x3f, sizeof(dist)); 
```

**为什么要用 `0x3f`？**
1.  **数值够大**：`0x3f3f3f3f` 大约是 $1.06 \times 10^9$（10亿多），对于大多数题目来说足够充当“无穷大”了。
2.  **防止溢出（最重要）**：
    *   如果你用 `INT_MAX` (约 20亿)，做松弛操作 `dist[u] + weight` 时，如果 `dist[u]` 已经是 `INT_MAX`，再加上一个正数 `weight`，就会**爆 `int` 变成负数**，导致算法错误。
    *   用 `0x3f3f3f3f`，即使加上一个普通的权值，结果也就是 20亿左右，依然在 `int` 范围内，非常安全！

**同理，如果求最大值需要初始化为“负无穷”：**
```cpp
memset(dp, 0xc0, sizeof(dp)); // 0xc0c0c0c0 约为 -10^9
```

---

### 4. 常见坑点总结 ⚠️

1.  **第三个参数是字节数，不是元素个数！**
    *   ❌ 错：`memset(a, 0, n);` （只清空了前 n 个**字节**，大概是前 n/4 个 int）
    *   ✅ 对：`memset(a, 0, sizeof(a));` （如果是全局数组/栈上数组）
    *   ✅ 对：`memset(a, 0, n * sizeof(int));` （如果是动态分配的数组/指针）

2.  **不要对 `vector` 使用 `memset`！**
    *   `vector` 是一个复杂的对象，里面有内部指针和结构。直接 `memset(&vec, ...)` 会破坏它的结构，导致 Crash。
    *   如果要清空 `vector`，用 `fill` 或者 `assign`：
        ```cpp
        vector<int> a(100);
        fill(a.begin(), a.end(), 0); // 推荐 std::fill
        // 或者
        a.assign(100, 0); 
        ```

3.  **不要对 double 数组用 `memset` 赋 0 以外的值**
    *   浮点数的二进制表示很复杂，`memset` 只能用来把 double 数组清 0。赋无穷大请用 `for` 循环或 `fill`。

---

### 推荐用法（背下来！）


1.  **清空数组 / 设为 0：**
    ```cpp
    memset(arr, 0, sizeof(arr));
    ```
2.  **设为 -1 (DP中表示未计算)：**
    ```cpp
    memset(dp, -1, sizeof(dp));
    ```
3.  **设为无穷大 (求最小值/最短路)：**
    ```cpp
    memset(dist, 0x3f, sizeof(dist));
    // 配合常量
    const int INF = 0x3f3f3f3f;
    if (dist[i] == INF) ...
    ```
