P1067 这道题是经典的**“模拟题”**。这类题目在算法上几乎没有难度（不需要 DP、不需要搜索），难点全在于**逻辑的缜密性**和**代码的整洁度**。很多新手会写出几十个 `if-else` 嵌套，最后自己都把自己绕晕了。

要想写得“简洁优美”，核心思想只有两个字：**解耦（Decoupling）**。

别把“符号”、“系数”、“x”、“指数”混在一起判断。我们把它们拆成**三个独立的流水线步骤**来处理。

### 核心思路拆解

对于每一项 $a_i x^i$，我们按顺序做以下决定：

1.  **判零**：如果是 0，直接 `continue`（跳过）。
2.  **符号处理**：
    *   如果是正数且不是第一项，输出 `+`。
    *   如果是负数，输出 `-`。
    *   （处理完符号后，后面只关心系数的**绝对值**，这样逻辑就统一了）。
3.  **系数处理**：
    *   如果是 1 且不是常数项（指数>0），**不输出**（省略1）。
    *   其他情况，输出绝对值。
4.  **指数处理**：
    *   指数 > 1：输出 `x^i`。
    *   指数 == 1：输出 `x`。
    *   指数 == 0：啥也不输出。

### 优雅的代码实现

你看，按照这个逻辑，代码就不需要嵌套深层 `if`，而是变成了平铺的结构：

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 1. 提速
    ios::sync_with_stdio(false);
    cin.tie(0);

    int n;
    cin >> n;

    // isFirst 用来标记是否是输出的第一项（处理正号用）
    bool isFirst = true;

    // i 代表当前的指数，从 n 遍历到 0
    for (int i = n; i >= 0; i--) {
        int a;
        cin >> a;

        // 【第一步：判零】系数为0直接跳过
        if (a == 0) continue;

        // 【第二步：处理符号】
        if (!isFirst && a > 0) cout << "+"; // 不是第一项的正数，要补+号
        if (a < 0) cout << "-";             // 负数永远要输出-号

        // 【第三步：处理系数】(注意这里用 abs 取绝对值)
        // 只有两种情况需要输出系数：
        // 1. 系数绝对值不为 1
        // 2. 是常数项 (i==0)，哪怕是 1 也要输出
        if (abs(a) != 1 || i == 0) {
            cout << abs(a);
        }

        // 【第四步：处理 x 和指数】
        if (i > 0) cout << "x";      // 只要不是常数项，就有 x
        if (i > 1) cout << "^" << i; // 只有指数大于 1，才显示 ^k

        // 既然已经输出过一项了，后面的就肯定不是第一项了
        isFirst = false;
    }

    return 0;
}
```

### “代码美学”解析 🎨

1.  **`abs(a)` 的妙用**：
    *   很多同学会写 `if (a == 1) ... else if (a == -1) ...`。
    *   其实处理完符号（`+` 或 `-`）后，`1` 和 `-1` 在显示系数时的逻辑是完全一样的（都不显示）。
    *   用 `abs(a)` 可以把正负数逻辑合并，代码量直接减半。

2.  **`bool isFirst` 标记**：
    *   专门用来解决“第一项如果是正数不能有 `+` 号”这个问题。这是处理行首空格/符号的通用技巧。

3.  **流水线结构**：
    *   符号 -> 系数 -> x -> 指数。
    *   互不干扰，逻辑清晰。
