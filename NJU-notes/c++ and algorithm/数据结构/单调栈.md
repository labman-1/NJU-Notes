单调栈是一种**巧妙利用栈维护元素单调性**的数据结构，常用于快速查找每个元素“左/右第一个比它大/小”的元素。下面从原理到代码，系统拆解。

---

## 一、核心原理

单调栈的核心是：**在栈中始终维持元素的单调性（递增或递减）**。当新元素入栈时，如果它破坏了单调性，就不断弹出栈顶元素，直到恢复单调。这个“弹出”的动作，恰好揭示了弹出的元素与当前新元素之间的关系。

### 以单调递减栈（栈底到栈顶递减）为例：
- **入栈规则**：当新元素 `x` 大于栈顶时，弹出栈顶，直到 `x` 可以放入且保持递减。
- **弹出的含义**：
  - 对于被弹出的元素 `top`，**`x` 是 `top` 右侧第一个比它大的元素**。
  - 弹出后新的栈顶（如果存在）是 `top` **左侧第一个比它大的元素**。

因此，通过单调递减栈，我们可以在 \(O(n)\) 时间内一次性得到每个元素的 **左右第一个更大值**。  
类似地，**单调递增栈** 可以得到每个元素的 **左右第一个更小值**。

---

## 二、适用场景

只要问题能转化为“找边界”或“找第一个比自己大/小的元素”，就可以尝试单调栈。

| 典型问题 | 说明 |
|----------|------|
| **下一个更大元素** (Next Greater Element) | 找每个元素右边第一个比它大的值 |
| **每日温度** (Daily Temperatures) | 找距离最近的下一个更高温度的天数差 |
| **柱状图中最大的矩形** (Largest Rectangle in Histogram) | 对每个高度，找左右第一个比它低的柱子 |
| **接雨水** (Trapping Rain Water) | 可以用单调递减栈维护凹槽 |
| **子数组最小值/最大值之和** | 需要找到每个元素作为最小值/最大值的影响范围 |
| **最大子矩阵** | 转化为直方图后用柱状图最大矩形解法 |
| **移除 K 位数字** (Remove K Digits) | 贪心 + 单调栈使结果最小 |

## 三、使用技巧

1. **选择递增还是递减**  
   - 要找**右边第一个比自己大** → 用**递减栈**（遇到更大的就弹出）。  
   - 要找**右边第一个比自己小** → 用**递增栈**（遇到更小的就弹出）。  
   记忆：**栈的单调方向与要找的“更大/更小”相反**。

2. **栈中存索引还是值？**  
   一般**存索引**，因为可以通过原数组取值，同时还能计算距离（如天数差、宽度等）。

3. **何时记录答案**  
   - **弹出时**：对于被弹出的元素，当前新元素就是它的“右侧第一个满足条件的元素”。  
   - **入栈后**：有时需要在遍历结束后，栈中剩余元素的右侧没有满足条件的元素，设为 -1 或特殊值。

4. **处理循环数组**  
   - 遍历两遍（`i = 0; i < 2*n`），下标取模 `i % n`，模拟环。

5. **处理重复元素**  
   - 需要明确题目要求是“严格大于”还是“大于等于”，相应地调整弹出条件（`<` 或 `<=`）。

## 四、C++ 代码示例

### 1. 每日温度（LeetCode 739）
求对于每一天，至少需要等待多少天才能看到一个更高温度。
```cpp
vector<int> dailyTemperatures(vector<int>& T) {
    int n = T.size();
    vector<int> res(n, 0);
    stack<int> st;  // 递减栈，存索引
    for (int i = 0; i < n; ++i) {
        while (!st.empty() && T[i] > T[st.top()]) {
            int prev = st.top(); st.pop();
            res[prev] = i - prev;   // 距离
        }
        st.push(i);
    }
    return res;
}
```
**解释**：栈中温度保持递减，遇到更高温时，弹出栈顶并计算天数差。

### 2. 下一个更大元素 I（LeetCode 496）
求 `nums1` 中每个元素在 `nums2` 中的下一个更大元素。
```cpp
vector<int> nextGreaterElement(vector<int>& nums1, vector<int>& nums2) {
    unordered_map<int, int> mp;  // 值 -> 下一个更大值
    stack<int> st;  // 递减栈，存值
    for (int x : nums2) {
        while (!st.empty() && x > st.top()) {
            mp[st.top()] = x;
            st.pop();
        }
        st.push(x);
    }
    vector<int> ans;
    for (int x : nums1)
        ans.push_back(mp.count(x) ? mp[x] : -1);
    return ans;
}
```

### 3. 柱状图中最大的矩形（LeetCode 84）
给定柱子高度数组，求能勾勒出的最大矩形面积。
```cpp
int largestRectangleArea(vector<int>& heights) {
    heights.push_back(0);  // 哨兵，保证最后全部弹出
    int n = heights.size();
    stack<int> st;  // 递增栈，存索引
    int max_area = 0;
    for (int i = 0; i < n; ++i) {
        while (!st.empty() && heights[i] < heights[st.top()]) {
            int h = heights[st.top()]; st.pop();
            int left = st.empty() ? -1 : st.top();
            int width = i - left - 1;
            max_area = max(max_area, h * width);
        }
        st.push(i);
    }
    return max_area;
}
```
**解释**：当遇到更矮的柱子时，以栈顶柱子为高的矩形右边界就是当前位置，左边界是弹出后的新栈顶，由此计算宽度。

## 五、总结

- 单调栈**将 O(n²) 的双重循环优化到 O(n)**，核心在于利用单调性维护候选集合，并在弹出时完成关键信息的记录。
- 解决该类问题的**黄金三步**：
  1. 确定要找的是“更大”还是“更小”。
  2. 选择对应的栈单调方向。
  3. 在入栈/出栈时计算所需信息（距离、面积、值等）。

```C++
#include<bits/stdc++.h>
using namespace std;
int n;
constexpr int MAX=80005;
int cow[MAX];
int main(){
	ios::sync_with_stdio(false);cin.tie(0);
	cin>>n;
	stack<int>st;
	long long ans=0;
	for(int i = 1;i<=n;i++){
		cin>>cow[i];
		while(!st.empty()&&st.top()<=cow[i]){
			st.pop();
		}
		ans+=st.size();
		st.push(cow[i]);
	}
	cout<<ans;
	return 0;
}

```
