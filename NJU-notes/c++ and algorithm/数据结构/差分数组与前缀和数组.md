**前缀和是用元数据求元与元之间的并集关系，而差分则是根据元与元之间的逻辑关系求元数据，是互逆思想**
**差分数组（Difference Array）** 是算法竞赛中处理“区间修改”问题的神器。如果说 **前缀和** 是用来快速“查”的，那 **差分** 就是用来快速“改”的。它们俩是一对互逆的“双胞胎”。
### 1. 痛点是什么？😫

假设你有一个数组 $A$，一开始全是 0。
现在老师给你下达了 $M$ 个指令（$M=10^6$），每个指令是：“请把从第 $L$ 个到第 $R$ 个数，全部加上 $V$”。
最后问你：修改完后，第 $i$ 个数是多少？

*   **暴力做法**：写个 `for (k = L; k <= R; k++) A[k] += V;`。
    *   一次修改是 $O(N)$。
    *   $M$ 次修改就是 $O(M \times N)$。
    *   如果 $N, M$ 都是 $10^6$，那就是 $10^{12}$ 次运算，**直接 TLE（超时）**。

*   **差分做法**：我们能不能不要傻傻地一个个加，而是像插旗子一样，**只标记开头和结尾**？这样一次修改只要 $O(1)$！

---

### 2. 差分的原理：前缀和的逆运算 🔄

我们定义一个差分数组 $D$。
它的定义非常简单：**$D[i] = A[i] - A[i-1]$**。
也就是：**$D[i]$ 记录了 $A[i]$ 比前一个数大了多少（变化量）。**

**神奇的性质：**
如果你知道了差分数组 $D$，怎么还原回原数组 $A$？
*   $A[1] = D[1]$
*   $A[2] = A[1] + D[2] = D[1] + D[2]$
*   ...
*   **$A[i] = \sum_{k=1}^i D[k]$**（$D$ 的前缀和就是 $A$！）

---

### 3. 核心操作：如何 $O(1)$ 修改区间？⚡

这是差分的灵魂所在。
如果你想让 $A$ 数组在区间 $[L, R]$ 内的所有数都加上 $V$。

我们来看看差分数组 $D$ 会发生什么变化：

1.  **对于 $A[L]$**：它比 $A[L-1]$ 突然多增加了 $V$（因为 $A[L]$ 被加了，$A[L-1]$ 没加）。
    *   所以：**$D[L] += V$**。
2.  **对于中间的数（比如 $A[L+1]$）**：它加了 $V$，它前面的 $A[L]$ 也加了 $V$。它俩的差值变了吗？没变！
    *   所以：中间的差分值不用动。
3.  **对于 $A[R+1]$**：它没加 $V$，但它前面的 $A[R]$ 加了 $V$。所以它相对于前一个数，变小了 $V$。
    *   所以：**$D[R+1] -= V$**。

**结论（背诵全文）：**
要把区间 $[L, R]$ 全部加上 $V$，只需要两步操作：
1.  **`D[L] += V`** （在起点打个标记：从此开始，地势升高 V）
2.  **`D[R+1] -= V`** （在终点的**下一位**打个标记：从此开始，地势恢复）

### 4. 代码模板 📝

```cpp
#include <vector>
#include <iostream>
using namespace std;

// 假设题目数据范围
const int MAXN = 1000005;
long long diff[MAXN]; // 差分数组，开 long long 防溢出
long long a[MAXN];    // 原数组

int main() {
    int n, m;
    cin >> n >> m;

    // 1. 初始化（如果原数组一开始不是0，也可以构建差分）
    // 假设一开始全是0，那 diff 也是全是 0，啥都不用做

    // 2. 处理 m 次区间修改
    for (int i = 0; i < m; i++) {
        int l, r, v;
        cin >> l >> r >> v;
        
        // 核心操作 O(1)
        diff[l] += v;
        diff[r + 1] -= v; 
    }

    // 3. 还原数组 (求前缀和) O(N)
    // 此时 diff 里的数据是离散的，我们需要把它变成连续的实际值
    long long current_val = 0;
    for (int i = 1; i <= n; i++) {
        current_val += diff[i]; // 一路累加
        a[i] = current_val;     // 还原得到 A[i]
        // 或者直接 diff[i] += diff[i-1] 原地还原
    }

    return 0;
}
```
