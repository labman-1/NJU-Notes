## Iteration
To support the enhanced for loop:
- Add ana ierator() method to your class that returns an Iterator<\T>.
- The Iterator<\T> returned should have a useful hasNext() and next() method.
- Add implements Iterable<\T> to the line defining your class.
## this
- Unlike Python, where self is mandatory, using this is not mandatory.
- If there's ever name conflict where *a local variable has the same name as an instance variable*, you **must** use this if you want to access the instance variable.
