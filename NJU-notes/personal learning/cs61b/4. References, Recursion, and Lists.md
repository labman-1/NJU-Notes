## The Golden Rules of Equals (GRoE)
Given variables y and x:
- y = x **copies** all the bits from x into y.
Passing parameters obeys the same rule:Simply **copy the bits** (pass by value) to the new scope.
## Reference Types 
There are 8 primitive types in Java:
-  byte, short, **int**, long float, **double**, boolean, char
Everything else, including arrays, is a **reference type**. 
## Invariants
An invariant is a condition that guaranteed to be true during code execution(assuming there are no bugs in your code).
Invariansts make it easier to reason about code:
- Can assume they are true to simplify code(e.g. addLast doesn't need to worry about nulls).
- Must ensure that methods preserve invariants.
