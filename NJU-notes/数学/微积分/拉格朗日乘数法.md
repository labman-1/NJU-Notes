
标准结构
$$ \begin{cases} L_x' = 0 \ L_y' = 0 \ L_z' = 0 \ g(x,y,z) = 0 \end{cases} $$

常用技巧
对称结构 → 设比值
若 $L_x'$、$L_y'$、$L_z'$ 形式高度对称（如均含 $yz$、$xz$、$xy$），可设 $\dfrac{L_x'}{L_y'} = \dfrac{x}{y}$ 等比例关系简化。

乘变量消项
将 $L_x' = 0$ 两边同乘 $x$，得 $x L_x' = 0$，常可消去含 $yz$ 的交叉项（类似处理 $y L_y'$、$z L_z'$）。

相减消元
对 $L_x' = 0$ 与 $L_y' = 0$ 两式作差：$(L_x' - L_y') = 0$，可消去相同项（如公共的 $\lambda$ 项或常数项）。

代入约束求解
将前几步得到的关系（如 $x = y$、$z = 2x$）代入约束方程 $g(x,y,z) = 0$，解出具体数值解。
