奠基：当n=1时，显然R时自反的。则$(a,a)\in R$.

则$R^{k+1}$也是自反的。

void warshall(int n, vector<vector<int>>& M) {
    for (int k = 0; k < n; k++) { // 中转站
        for (int i = 0; i < n; i++) { // 起点
            if (M[i][k]) { 
                for (int j = 0; j < n; j++) { // 终点
                    M[i][j] = M[i][j] | M[k][j];
                }
            }
        }
    }
}

(a)
限制条件为$m_{i,j}=m_{j,i}$
对称关系数为$2^n\cdot 2^{\frac{n^2-n}{2}}=2^{\frac{n^2+n}{2}}$
(b)
限制条件为$m_{i,j}与m_{j,i}$中之多有一个为1。
反对称关系数为$3^{\frac{n^2-n}{2}}\cdot 2^n$
(c)
在反对称的基础上加上$m_{i,i}=0$的限制条件。
强反对称的关系数为$1^n\cdot3^{\frac{n^2-n}{2}}=3^{\frac{n^2-n}{2}}$
(d)
限制条件为$m_{i,i}=0$
则反自反的关系数为$2^{n^2-n}$
(e)
限制条件为$m_{i,i}=1$且$m_{i,j}=m_{j,i}$
对应的关系数为$1^n\cdot2^{\frac{n^2-n}{2}}=2^{\frac{n^2-n}{2}}$
(f)
限制条件为对角线不全相同。
对应的关系数为$(2^n - 2) \cdot 2^{n^2-n}$
