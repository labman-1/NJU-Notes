(b)证明：
设 $d^{(p)}(v_j, v_k)$ 表示最外层循环执行完 $i = p$ 时，算法中 $d(v_j, v_k)$ 的值。
同时，定义数学上的最优值 $D(v_j, v_k, p)$ 为：从 $v_j$ 到 $v_k$ 且路径上中间顶点都属于集合 $\{v_1, v_2, \dots, v_p\}$ 的所有路径中的最短距离。

我们要证明，对任意的 $0 \le p \le n$ 且对任意的顶点对 $(v_j, v_k)$，均有：
$$d^{(p)}(v_j, v_k) = D(v_j, v_k, p)$$

下面对 $p$ 施加数学归纳法。
#### 1. 奠基步：
当 $p = 0$ 时，允许使用的中间顶点集合为 $\emptyset$。
此时，从 $v_j$ 到 $v_k$ 的路径如果不能有任何中间顶点，则只能是直接连接它们的边 $(v_j, v_k)$。
根据定义，此时的最短距离为：
$$D(v_j, v_k, 0) = w(v_j, v_k)$$
而在算法的初始化阶段，我们有：
$$d^{(0)}(v_j, v_k) = w(v_j, v_k)$$
故当 $p = 0$ 时， $d^{(0)}(v_j, v_k) = D(v_j, v_k, 0)$ 成立。

#### 2. 归纳假设：
假设当 $p - 1$ 时结论成立。即对于任意的顶点对 $(v_y, v_z)$，均有：
$$d^{(p-1)}(v_y, v_z) = D(v_y, v_z, p-1)$$

#### 3. 归纳步骤：
现在考虑外层循环执行到 $i = p$ 结束时。我们需要证明：
$$d^{(p)}(v_j, v_k) = D(v_j, v_k, p)$$

对于只允许使用 $\{v_1, \dots, v_p\}$ 作为中间顶点的 $v_j$ 到 $v_k$ 的最短路径 $P$，有且仅有两种互斥的情况：
*   **情况一：路径 $P$ 不经过顶点 $v_p$**
    此时 $P$ 的所有中间顶点都属于 $\{v_1, \dots, v_{p-1}\}$。
    根据归纳假设，此最短路径的长度为 $D(v_j, v_k, p-1)$。
    
*   **情况二：路径 $P$ 经过顶点 $v_p$**
    因为图中的所有边权均为正数，最短路径 $P$ 必定是简单路径，从而 $v_p$ 作为中间节点在 $P$ 中出现且仅出现一次。
    因此，我们可以将 $P$ 拆分为 $P_1$（从 $v_j$ 到 $v_p$）和 $P_2$（从 $v_p$ 到 $v_k$）两部分：
    由于 $v_p$ 是 $P_1$ 的终点和 $P_2$ 的起点，它不能是这两段子路径的**中间节点**。因此， $P_1$ 与 $P_2$ 的中间节点均属于 $\{v_1, \dots, v_{p-1}\}$。
    根据最短路径的子路径最优性原理，$P_1$ 必须是 $v_j$ 到 $v_p$ 且中间节点在 $\{v_1, \dots, v_{p-1}\}$ 中的最短路径，则由归纳假设知，$P_{1}$长度为 $D(v_j, v_p, p-1)$；同理， $P_2$ 的长度为 $D(v_p, v_k, p-1)$。
    故该情况下最短路径长度为 $D(v_j, v_p, p-1) + D(v_p, v_k, p-1)$。

综合上述两种情况，根据最优性原理，有：
$$D(v_j, v_k, p) = \min\Big( D(v_j, v_k, p-1), \; D(v_j, v_p, p-1) + D(v_p, v_k, p-1) \Big)$$

而在算法的 $i = p$ 这次迭代中，状态转移方程执行的是：
$$d^{(p)}(v_j, v_k) = \min\Big( d^{(p-1)}(v_j, v_k), \; d^{(p-1)}(v_j, v_p) + d^{(p-1)}(v_p, v_k) \Big)$$

根据归纳假设，将 $d^{(p-1)}$ 全部替换为 $D(\cdot, \cdot, p-1)$：
$$d^{(p)}(v_j, v_k) = \min\Big( D(v_j, v_k, p-1), \; D(v_j, v_p, p-1) + D(v_p, v_k, p-1) \Big)$$

对比两式可知：
$$d^{(p)}(v_j, v_k) = D(v_j, v_k, p)$$
归纳步骤得证。

#### 结论：
由数学归纳法可知，对于任意 $0 \le p \le n$，当 Floyd 算法最外层循环执行完 $i = p$ 时， $d(v_j, v_k)$ 均表示从 $v_j$ 到 $v_k$ 且路径上中间顶点均在 $\{v_1, \dots, v_p\}$ 中的最短通路长度。 $\quad\blacksquare$

Dijkstra 算法的核心假设是：如果顶点 u 是目前所有未确定最短路的顶点中 L(u) 最小的那一个，那么当前记录的 L(u) 已经必定是从源点到 u 的绝对最短距离。
算法之所以能做出这个确信，是因为它认为：任何其他可能到达 u 的路径，都必然要先经过其他未确定的顶点 v。而由于 L(v) 已经大于 L(u)，且图中所有边权均为非负数，那么从 v 延伸到 u 的路径长度只可能更大，绝不可能比当前的 L(u) 更小。一旦边权可以为负数，这个假设就不成立了，进而算法失效。
**反例**：构建一个只有 3 个顶点的有向无环图。
设顶点集合为 {A, B, C}，其中 A 为源点。
有向边及权值如下：
- A 到 B，权值为 2
- A 到 C，权值为 5
- C 到 B，权值为 -10
这是一个完全没有环的简单图。我们来模拟 Dijkstra 算法的执行过程：
#### 初始状态
- L(A) = 0
- L(B) = 无穷大
- L(C) = 无穷大
- S = 空集
#### 第一轮迭代
1. 挑选未确定顶点中 L 值最小的一个，即源点 A。
2. 将 A 加入集合 S。此时 S = {A}。
3. 从 A 出发松弛邻接点：
   - L(B) 更新为 min{无穷大, 0 + 2} = 2
   - L(C) 更新为 min{无穷大, 0 + 5} = 5
#### 第二轮迭代
1. 在未确定顶点 B 和 C 中，挑选 L 值最小的。因为 L(B) = 2 小于 L(C) = 5，算法挑选了 B。
2. 将 B 加入集合 S。此时 S = {A, B}。
3. 算法从此确信：从 A 到 B 的最短路长度就是 2，后续绝不会再对其进行更新。
4. 从 B 出发没有邻接点可以松弛。

#### 第三轮迭代
1. 挑选剩余未确定的顶点中 L 值最小的，即 C。
2. 将 C 加入集合 S。此时 S = {A, B, C}。
3. 从 C 出发，邻接点为 B。
   - 此时算法本应计算得到一条新路径：A 到 C 再到 B，长度为 5 + (-10) = -5。
   - 但是，因为 B 已经在之前的步骤中被强行加入了集合 S，且被标记为已确定，所以标准 Dijkstra 算法此时绝不会再去更新 L(B)。
4. 算法结束。

#### 最终输出结果
- Dijkstra 算法给出从 A 到 B 的最短距离为 2。
- 但实际上，真实的最短路径是 A 到 C 再到 B，其长度为 -5。
- 算法给出了错误的结果。

## 反驳
该命题不成立。我们可以通过构建两张不同的图 $G_1$ 与 $G_2$，使它们具有相同的顶点集和完全相同的点对间最短路距离，从而证明无法唯一地重构出原图。
### 构造反例
考虑一个顶点集合 $V = \{a, b, c\}$。我们分别构建两张简单、连通且边权均为正数的图 $G_1$ 和 $G_2$。
#### 图 1 的构造
设无向图 $G_1 = (V, E_1, w_1)$：
- 边集为：$E_1 = \{(a, b), (b, c)\}$
- 边权为：
  - $w_1(a, b) = 1$
  - $w_1(b, c) = 1$

在 $G_1$ 中，所有顶点对之间的最短路距离分别为：
- $d_1(a, b) = 1$
- $d_1(b, c) = 1$
- $d_1(a, c) = d_1(a, b) + d_1(b, c) = 1 + 1 = 2$

#### 图 2 的构造
设无向图 $G_2 = (V, E_2, w_2)$：
- 边集为：$E_2 = \{(a, b), (b, c), (a, c)\}$
- 边权为：
  - $w_2(a, b) = 1$
  - $w_2(b, c) = 1$
  - $w_2(a, c) = 2$

在 $G_2$ 中，所有顶点对之间的最短路距离分别为：
- $d_2(a, b) = 1$
- $d_2(b, c) = 1$
- $d_2(a, c) = \min\{w_2(a, c), \; d_2(a, b) + d_2(b, c)\} = \min\{2, 1+1\} = 2$

### 分析
1. 图 $G_1$ 和 $G_2$ 均符合“权值为正的简单连通图”这一前提条件。
2. 它们所有顶点对之间的最短路距离完全等价，即：
   - $d_1(a, b) = d_2(a, b) = 1$
   - $d_1(b, c) = d_2(b, c) = 1$
   - $d_1(a, c) = d_2(a, c) = 2$
1. 然而，图 $G_1$ 与图 $G_2$ 的边数和拓扑结构完全不同：$G_1$ 仅包含 2 条边，而 $G_2$ 包含 3 条边。两张图在图同构意义下也是非同构的（边数不同）。
因此，仅仅在已知任意两点间最短路长度的前提下，我们无法唯一地构建（重构）出原图 $G$。原命题不成立。$\quad\blacksquare$
