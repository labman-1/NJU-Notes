# 1
证明：
设简单二部图 $G = (V, E)$ 的顶点集 $V$ 可以划分为两个互不相交的独立集 $V_1$ 和 $V_2$，即 $V = V_1 \cup V_2$ 且 $V_1 \cap V_2 = \emptyset$。

设两个独立集的顶点数分别为 $v_1 = |V_1|$ 和 $v_2 = |V_2|$。根据定义，有：
$v_1 + v_2 = |V|$

因为 $G$ 是简单二部图，所以图中的每一条边都必须连接 $V_1$ 中的一个顶点与 $V_2$ 中的一个顶点，且图中不存在重边和自环。

由于 $V_1$ 与 $V_2$ 之间可能存在的最大边数在完全二部图 $K_{v_1, v_2}$ 时达到，此时每一对跨越独立集的顶点都由一条边连接。因此，任意简单二部图 $G$ 的边数 $E$ 必须满足：$E \le v_1 v_2$

根据基本不等式，对于非负实数 $v_1$ 和 $v_2$，有：
$v_1 v_2 \le \left(\frac{v_1 + v_2}{2}\right)^2$

将 $v_1 + v_2 = V$ 代入上式中，可得：
$v_1 v_2 \le \left(\frac{V}{2}\right)^2 = \frac{V^2}{4}$

因此，我们有：
$E \le v_1 v_2 \le \frac{V^2}{4}$

证毕。$\quad\blacksquare$

# 2
要在 $2n$ 个孤立顶点中加入 $n$ 条边，使得到的图包含这 $2n$ 个顶点的完美匹配，这意味着新加入的这 $n$ 条边之间必须互不相邻（即任何两条边不共享端点）。
因此，问题等价于 在 $2n$ 个有标记的顶点中，进行两两无序配对，共有多少种不同的配对方案。
采用以下步骤选择配对方案：
1. 任意选择一个尚未配对的顶点，此时还剩下 $2n-1$ 个未配对的顶点，因此为该顶点选择配对对象有 $2n-1$ 种选择。
2. 在剩下的 $2n-2$ 个未配对顶点中，再任意选择一个，此时还剩下 $2n-3$ 个未配对的顶点，因此选择配对对象有 $2n-3$ 种选择。
3. 依此类推，每次配对都会使未配对顶点数减少 2。
4. 最终，总的配对方案数为：
   $(2n-1) \times (2n-3) \times \dots \times 3 \times 1 = (2n-1)!!$

综上所述，总共能得到 $(2n-1)!!$ 种不同的完美匹配。 $\quad\blacksquare$
