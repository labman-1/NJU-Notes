# Problem 2
（a）当x=2时，$x^2=2$，则原命题为真
（b）$\because \forall x(x^2>=0)$
$\therefore \forall x(x^2\ne-1)$
即原命题为假
（c）$\because \forall x(x^2+2>=2)$
$\therefore \forall x(x^2+2>=1)$
即原命题为真
（d）当x=1时，$x^2=x$，则原命题为假。
