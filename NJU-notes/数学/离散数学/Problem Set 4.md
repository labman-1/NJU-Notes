# Problem 1
证明：（归谬法）假设$\exists 正整数n=4m+3(m\in \mathbb{N})可以写成两个整数的平方和$，设这两个整数为a，b，则有$a^2+b^2=4m+3$。由4m+3为奇数得$a^2+b^2$为奇数，则a和b一奇一偶，不妨设a为奇数，b为偶数，进而可设a=2k+1，b=2l（k和l为整数）。代回原式有$(2k+1)^2+(2l)^2=4(k^2+k+l^2)+1=4m+3$，移项可得$k^2+k+l^2-m=\frac{1}{2}$，这与k，l，m均为整数矛盾。所以假设不成立。Q.E.D.
