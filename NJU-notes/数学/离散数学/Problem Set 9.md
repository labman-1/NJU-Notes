# P3
证明：命题：平面上过同一点的 n 条直线将平面分为 2n 个区域。
我们用数学归纳法证明该命题。
 **（1）基例：n=1**

当只有一条直线时，它把平面分成 2 个区域，而2=2⋅1，命题成立。

 **（2）归纳假设**

假设当有 k 条直线都经过同一点 P 时，它们将平面分成2k个区域。

 **（3）归纳步骤**
现在加入第 k+1 条直线 L，并且它也经过点 P。
由于所有直线都经过同一点 P，因此：
- 新直线 L 与已有的 k 条直线**只有一个交点**（即 P）
    
- 不会产生新的交点
    
- 原来的区域结构是由 k 条直线形成的 2k 个扇形区域
    

因此，新直线 L 只会把原来的两个扇形区域各切成两半，从而：

- 每个被切开的区域增加 1 个新区域
    
- 总共增加 2 个区域
    
于是加入 L 后的区域总数为：2k+2=2(k+1)，这正是命题在 n=k+1 时的结论。
**（4）结论**
由数学归纳法，命题对所有正整数 n 都成立。

# P5
## 1.
1. 将建筑物按位置从小到大排序。
2. 找到最左边且未被覆盖的建筑物 $x_{first}$。
3. 在 $x_{first} + 1$ 的位置放置一座基站塔（这样能保证覆盖到 $x_{first}$，且向右延伸得最远）。
4. 移除所有已被这座新塔覆盖的建筑物。
5. 重复上述步骤直到所有建筑物被覆盖。

## 2.
命题 $P(k)$：如果一组建筑物最少可以用 $k$ 座塔覆盖，那么贪心算法也一定能用 $k$ 座塔覆盖它们。
1. 奠基步：$k=1$
如果一组建筑物最少可以用 1 座塔覆盖，说明这组建筑物中最左边 $x_{min}$ 和最右边 $x_{max}$ 的距离 $|x_{max} - x_{min}| \le 2$。
按照贪心算法，第一座塔放在 $g_1 = x_{min} + 1$处。
因为 $x_{max} \le x_{min} + 2$，所以 $x_{max} \le g_1 + 1$。
这意味着所有的建筑物都在 $[g_1-1, g_1+1]$ 范围内，贪心算法也只用了 1 座塔。$P(1)$ 成立。
2. 归纳假设 
假设对于任何最少只需 $k$ 座塔覆盖的建筑物集合，贪心算法也只需 $k$ 座。
3. 归纳步骤
考虑一个最少需要 $k+1$ 座塔覆盖的集合 $X$。
设某种最优解放置的塔为 $\{o_1, o_2, \dots, o_{k+1}\}$，其中 $o_1$ 是负责覆盖最左侧建筑物 $x_{first}$ 的那座塔。
易知最优解的 $o_1$ 必须满足 $o_1 \le x_{first} + 1$(否则$o_1$无法覆盖第一座建筑物)。
而贪心算法放的第一座塔 $g_1 = x_{first} + 1$，所以 $o_1 \le g_1$。
则被 $o_1$ 覆盖的建筑物集合 $S_o$ 必然是被 $g_1$ 覆盖的集合 $S_g$ 的子集（即 $S_o \subseteq S_g$）。
这意味着，当我们用掉贪心算法的第一座塔后，剩下未覆盖的建筑物集合 $X' = X \setminus S_g$。
因为 $X \setminus S_g \subseteq X \setminus S_o$，而 $X \setminus S_o$ 显然可以被剩下的 $k$ 座最优塔 $\{o_2, \dots, o_{k+1}\}$ 覆盖。
所以，$X'$ 这个集合最少只需要 $k$ 座（甚至更少）塔就能覆盖。
根据归纳假设，贪心算法处理 $X'$ 集合也只需要 $k$ 座塔。
加上贪心算法第一步用掉的 1 座塔，总数为 $1 + k$。
因此，$P(k+1)$ 成立。
4. 结论
由数学归纳法可知，对于任意 $k$，贪心算法使用的塔数均等于最优解的塔数。

# P8
证明：
命题 $P(n)$：对于长度为 $n$ 的任意 0/1 序列，满足 $N(01) - N(10) \in \{-1, 0, 1\}$。
1. 奠基：$n=1$
序列只有 0 或 1。
此时 $N(01) = 0, N(10) = 0$。
$N(01) - N(10) = 0$，属于 $\{-1, 0, 1\}$，命题成立。
2. 归纳假设 
假设对于长度为 $k$ 的任意序列 $S$，其 $N(01) - N(10) = V \in \{-1, 0, 1\}$ 成立。
3. 归纳步骤 (
考虑长度为 $k+1$ 的序列 $S'$，其可以看作在长度为 $k$ 的序列 $S$ 后面添加了一个字符 $x$。
设 $S$ 的最后一个字符为 $last$，$S'$ 相比 $S$ 多出的部分取决于 $last$ 和 $x$ 的关系：
*   情况 A：$last = x$（例如 `...00` 或 `...11`）
    此时没有产生新的 `01` 或 `10` 块。
    $N(01)' = N(01)$， $N(10)' = N(10)$。
    差值不变，$P(k+1)$ 成立。
*   情况 B：$last = 0$ 且 $x = 1$（产生了一个新的 `01`）
    此时 $N(01)' = N(01) + 1$，$N(10)' = N(10)$。
    新的差值 $V' = V + 1$。
    要证明 $V+1 \le 1$，需要知道 $V$ 原本是多少。
    既然 $last=0$，说明序列 $S$ 是以 $0$ 结尾的。
    1. 若 $S$ 以 $0$ 开头，以 $0$ 结尾，则 $V=0 \implies V'=1$。
    2. 若 $S$ 以 $1$ 开头，以 $0$ 结尾，则 $V=-1 \implies V'=0$。
    无论哪种，$V'$ 依然在 $\{-1, 0, 1\}$ 中。

*   情况 C：$last = 1$ 且 $x = 0$（产生了一个新的 `10`）
    此时 $N(01)' = N(01)$，$N(10)' = N(10) + 1$。
    新的差值 $V' = V - 1$。
    同理，$S$ 以 $1$ 结尾，其 $V$ 只能是 $0$（1开头）或 $1$（0开头），减 1 后依然符合要求。
4. 结论
由数学归纳法可知，差值永远不会超过 1。则命题P成立，则比P更弱的命题 (在任意长度有限的 0/1 序列中，字符串 01 至多比字符串 10 多出现 1 次) 成立.

	(i) 奠基：当n=1时，左式=$\sum_{k=1}^1k^1=1$, $右式=\frac{1*2}{2}=1$，左右两式相等。

	则当n=t+1时，$\sum_{k=1}^{t+1} k^1=\sum_{k=1}^t k^1+(t+1)=\frac{t(t+1)}{2}+(t+1)=\frac{(t+1)(t+2)}{2}$,原式成立。
	由数学归纳法，$\forall n\in\mathbb{N}_+,\sum_{k=1}^n k^1=\frac{n(n+1)}{2}$成立。
	(ii)奠基：当n=1时，左式=$\sum_{k=1}^1k^2=1$, $右式=\frac{1*2*3}{6}=1$，左右两式相等。

	由数学归纳法，$\forall n\in\mathbb{N}_+,\sum_{k=1}^n k^2=\frac{n(n+1)(2n+1)}{6}$成立。
(2)
根据二项式定理：
$$(k+1)^{m+1} = \binom{m+1}{0}k^{m+1} + \binom{m+1}{1}k^m + \binom{m+1}{2}k^{m-1} + \dots + \binom{m+1}{m+1}$$
整理得：
$$(k+1)^{m+1} - k^{m+1} = \binom{m+1}{1}k^m + \binom{m+1}{2}k^{m-1} + \dots + \binom{m+1}{m}k + 1$$
对上面的等式左、右两边同时从 $k=1$ 到 $n$ 求和：
左边：$\sum_{k=1}^n ((k+1)^{m+1} - k^{m+1})$。
即 $(2^{m+1}-1^{m+1}) + (3^{m+1}-2^{m+1}) + \dots + ((n+1)^{m+1}-n^{m+1}) = \mathbf{(n+1)^{m+1} - 1}$
右边：
$\binom{m+1}{1}\sum_{k=1}^n k^m + \binom{m+1}{2}\sum_{k=1}^n k^{m-1} + \dots + n$

如果把 $\sum_{k=1}^n k^j$ 记作 $S_j(n)$，则上式等价于：
$$(n+1)^{m+1} - 1 = \sum_{j=0}^{m} \binom{m+1}{j} S_j(n)$$
等价于
$$S_m(n) = \frac{1}{m+1} \left[ (n+1)^{m+1} - 1 - \sum_{j=0}^{m-1} \binom{m+1}{j} S_j(n) \right]$$

证明：
1. 奠基： 当m=0时，$S_0(n)=n$;由(1)知$S_1(n)=\sum_{k=1}^n k^1=\frac{n(n+1)}{2}, S_2(n)=\sum_{k=1}^n k^2=\frac{n(n+1)(2n+1)}{6}$,代入可知符合公式
2. 强归纳假设：假设对于所有 $j < m$，求和式 $S_j(n)$ 都是关于 $n$ 的 $j+1$ 阶多项式。
3.  归纳步骤：
    *   依据归纳步骤
    *   $(n+1)^{m+1}$ 本身是 $m+1$ 阶的。
    *   低阶多项式相加减，不会改变最高次幂。
    *   因此，$S_m(n)$ 必然也是一个 $m+1$ 阶的多项式。
4.  系数校验：
    注意到 $(n+1)^{m+1}$ 展开后的 $n^{m+1}$ 项系数是 $1$，分母是 $\binom{m+1}{1} = m+1$。
    所以 $S_m(n)$ 的最高次项必然是 $\frac{1}{m+1}n^{m+1}$。
