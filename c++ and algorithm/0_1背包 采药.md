# 0/1背包 采药

[https://www.luogu.com.cn/problem/P1048#ide](https://www.luogu.com.cn/problem/P1048#ide)  
原地dp  
注意点：正序 or 逆序？  
逆序才可保证用到的数据均为旧数据而没有重复采药。  
```cpp
#include <bits/stdc++.h>
#define T 1005
#define M 105
int d[T];
int times[M], value[M];
using namespace std;
int main() {
	ios::sync_with_stdio(false);
	cin.tie(0);
	int t, m;
	cin >> t >> m;
	for (int i = 1; i <= m; i++) cin >> times[i] >> value[i];
	for (int i = 1;i <= m;i++) {
		for (int j = t;j >=times[i];j--) {
			d[j] = max(d[j - times[i]] + value[i], d[j]);
		}
	}
	cout<<d[t];
	return 0;
}
```
