##### 关键优化  

●单边递归：这行 if (k <= j) ... else if (k >= i) ... 是整个算法的灵魂，它直接把平均复杂度从 O(Nlog⁡N)砍到了 O(N)。  
●随机化 Pivot：加了 srand 和 rand，虽然只是简单的随机，但在绝大多数 OI 评测机（通常是 Linux 环境）上，足以抵抗出题人的恶意卡常数据了。  
●IO 加速：保留了良好的 IO 习惯。
