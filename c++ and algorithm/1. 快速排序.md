# 1. 快速排序

[第k大的数](https://www.luogu.com.cn/problem/P1923)  

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[5000005];
void quick_sort(int a[],int low,int high) {
    if (low >= high) return;
    int i = low, j = high;
    int mid = (low + high) / 2;
    int pivot = a[mid];
    while (i<=j) {
        while (a[i]<pivot)i++;
        while (a[j] >pivot) j--;
        if (i<= j) {
            swap(a[i], a[j]);
            i++;
            j--;
        }
    }
    if (low<j) quick_sort(a, low, j);
    if (i<high)quick_sort(a, i, high);
}
int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    int n,k;
    cin >> n >> k;
    for (int i = 0;i < n;i++) {
        cin >>a[i];
    }

    quick_sort(a, 0, n - 1);
    cout << a[k];
    return 0;
}
```

上述快排可能会TLE，其实可以优化到O（n）  
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[5000005];
void quick_sort(int a[],const int& k,int low,int high) {
	if (low >= high) return;
	int i = low, j = high;
	int pivot = a[low + rand() % (high - low + 1)];
	while (i<=j) {
		while (a[i]<pivot)i++;
		while (a[j] >pivot) j--;
		if (i<= j) {
			swap(a[i], a[j]);
			i++;
			j--;
		}
	}
	// 只需要判断 k 落在左边还是右边
	if (k <= j) {
		quick_sort(a, k, low, j); // k 在左半边，只需要排左边
	}
	else if (k >= i) {
		quick_sort(a, k, i, high); // k 在右半边，只需要排右边
	}
	// 如果 k 在 j 和 i 之间，说明找到了（或者区间很小了），
	// 因为外层判断了 low >= high 会 return，所以这里不用特意写 else return
	
	
}
int main() {
	ios::sync_with_stdio(false);
	cin.tie(0);
	int n,k;
	cin >> n >> k;
	for (int i = 0;i < n;i++) {
		cin >>a[i];
	}
	srand(time(0));
 	quick_sort(a,k, 0, n - 1);
	cout << a[k];
	return 0;
}
```

##### 关键优化  

●单边递归：这行 if (k <= j) ... else if (k >= i) ... 是整个算法的灵魂，它直接把平均复杂度从 O(Nlog⁡N)砍到了 O(N)。  
●随机化 Pivot：加了 srand 和 rand，虽然只是简单的随机，但在绝大多数 OI 评测机（通常是 Linux 环境）上，足以抵抗出题人的恶意卡常数据了。  
●IO 加速：保留了良好的 IO 习惯。  
#### 关于 rand() 的小秘密  

在 Windows 本地环境下（Visual Studio 等），RAND\_MAX 只有 32767。如果你的  N=5000000，单纯的 rand() % N 其实覆盖不到整个数组，随机性会大打折扣。 但在 Linux 环境下（洛谷、Codeforces、NOI Linux），rand() 通常很大，你这样写没问题。 进阶写法（以后备用）： 如果想绝对稳健，可以使用 C++11 的 mt19937：  
```cpp
mt19937 rng(time(0));
int pivot = a[low + rng() % (high - low + 1)];
```
（现在不改也没事，这种练手题 rand() 够用了）  
#### 2\. 竞赛中的“作弊码”：nth\_element  

虽然题目让你练习分治，不让你用库函数。但作为 OIer，你必须知道 C++ STL 里已经内置了这个算法。 如果是在真正的考场上（不是做练习题），遇到求第k小，千万别手写 Quick Select（万一写挂了呢？），直接一行代码解决：  
```cpp
// 将数组 a 中第 k 小的元素放到 a[k] 的位置，
// 且 a[k] 左边的都比它小，右边的都比它大。
nth_element(a, a + k, a + n);
cout << a[k];
```
这东西底层实现就是刚刚写的快速选择，平均复杂度 O（n），而且常数极小。
