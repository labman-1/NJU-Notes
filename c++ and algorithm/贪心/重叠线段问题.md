# 重叠线段问题

[链接](https://www.luogu.com.cn/problem/P1803)  
注意点：考虑在给定区间内从左到右放置线段，自然希望线段右端点离区间右端点越远越好（以获得更多放东西的空间）  

```cpp
#include<bits/stdc++.h>
using namespace std;
struct c {
	int start, end;
	bool operator <(c &x) {
		if (this->end < x.end) { return true; }
		else return false;
	}
};
int main() {
	int n;
	cin >> n;
	vector<c>a(n);
	int ans=1;
	for (int i = 0;i < n;i++) {
		cin >> a[i].start >> a[i].end;
	}
	sort(a.begin(), a.end());
	int end1 = a.begin()->end;
	for (c x : a) {
		if (x.start >= end1) { 
			ans++; 
			end1 = x.end;
		}
	}
	cout << ans;
	return 0;
}
```
