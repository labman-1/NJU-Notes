### 一、本课程讲述的科学祖师是谁？根据什么？

**科学祖师是塞理斯（Thales，泰勒斯，约公元前624年—公元前564年）。**

根据：

- 塞理斯是古希腊最早的哲学学派——米利都学派（爱奥尼亚学派）的创始人。
- 他提出了人类第一个需要证明的数学定理（塞理斯定理）。
- 他提出"万物源于水"，被亚里斯多德认定为第一条哲学原理兼第一条科学原理。
- 罗素在《西方哲学史》中指出：哲学始于塞理斯。
- 爱因斯坦所感叹的从古希腊开始的理性传统，正是从塞理斯开始的。

### 二、亚里斯多德演绎推理形式逻辑体系的初始前提是什么？

亚里斯多德区别出三种初始前提（它们本身不可证明，但可知是真的）：

1. **公设（axioms）**——没有它们就不可能推理的原理
2. **定义（definitions）**——假定术语的意义
3. **假设（hypotheses）**——假定某种事物对应于某术语

### 三、欧几里德原本第一卷的（英文）具体内容是什么？

Euclid's Elements Book 1 consists of:

**Definitions, Postulates (Axioms), Common Notions, and Propositions (theorems and constructions)** about the foundations of geometry, covering congruent figures, parallel lines, and rectilinear figures.

英文具体内容包括（节选自课程文件）：

**Definitions:**

- Definition 1: A point is that which has no part.
- Definition 2: A line is breadthless length.
- Definition 3: The ends of a line are points.
- Definition 4: A straight line is a line which lies evenly with the points on itself.

**Five Postulates:**

1. A straight line segment can be drawn joining any two points.
2. Any straight line segment can be extended indefinitely in a straight line.
3. Given any straight line segment, a circle can be drawn having the segment as radius and one endpoint as center.
4. All right angles are congruent.
5. The parallel postulate.

**Five Common Notions:**

1. Things that are equal to the same thing are equal to each other.
2. If equals are added to equals, then the wholes are equal.
3. If equals are subtracted from equals, then the remainders are equal.
4. Things that coincide with one another are equal to one another.
5. The whole is greater than the part.

**Propositions:** 48 propositions, culminating in Proposition 47 — the Pythagorean theorem.

### 五、试给出欧几里德证明该命题时所用图形

根据课程文件4.8中的证明描述，欧几里德证明命题47所用的图形如下（文字描述）：

```
                    G
                   /|
                  / |
                 /  |
                /   |
               /    |
              B-----F
             /|      |
            / |      |
           /  |      |
          /   |      |
         /    |      |
        C-----K------A-----H
         \    |      |
          \   |      |
           \  |      |
            \ |      |
             \|      |
              D------E
              |      |
              L------+
```

具体构造：

- 直角三角形ACB，直角在A处
- 在三边BC、AB、CA上分别作正方形CBDE、BAGF、ACIH
- 从A点作平行于BD和CE的直线，在K和L处分别垂直于BC和DE
- 连接CF和AD，形成三角形BCF和BDA
- 该线将斜边上的正方形分成两个矩形，每个矩形的面积分别等于两条直角边上两个正方形之一的面积
- 最终证明：AB² + AC² = BC²

此为经典的风车图（windmill diagram）。

### 六、笛卡尔科学方法"四戒律"是什么？

笛卡尔在1637年发表《谈谈方法》（_Discourse on the Method_）中提出了四条科学方法戒律：

**中文版：**

1. 永远不接受任何我自己不清楚的真理，就是说要尽量避免鲁莽和偏见，只能是根据自己的判断非常清楚和确定，没有任何值得怀疑的地方的真理。
2. 可以将要研究的复杂问题尽量分解为多个比较简单的小问题，一个一个地分开解决。
3. 将这些小问题从简单到复杂排列，先从容易解决的问题着手。
4. 将所有问题解决后，再综合起来检验，看是否完全，是否将问题彻底解决了。

**英文版（The four precepts of Descartes' Method of Science）：**

1. Filter away all that may be in doubt.
2. Divide difficulties to as small pieces as necessary.
3. Start with the simplest problems.
4. Make Lists, Tables, Diagrams.

### 七、康德是如何区分笛卡尔几何与欧氏几何的？

康德将判断分为**分析判断（analytic judgment）**与**综合判断（synthetic judgment）**：

- **Analytic judgment**: a judgment whose truth can be determined just by analyzing the term in it.
- **Synthetic judgment**: a judgment whose truth requires looking outside of its terms.

康德以欧几里德原本第一卷命题32的证明过程为例，指出证明过程中用到延长线和辅助平行线——这些突然冒出来的线是在"空间"中画出来的。康德认为：空间是一切可能知觉的形式（Space is the form of all possible perceptions），空间是理智的一种纯粹先验直觉（Space is a pure a priori intuition of the intellect）。因此康德判定：

- **欧几里德几何**：**先验综合判断（a priori synthetic judgment）**
- **笛卡尔几何**：**分析判断（analytic judgment）**，其分析性质完全来自数学分析（mathematical analysis）

欧几里德几何训练人的推理能力，笛卡尔几何则因采用数学分析而成为实用技术。

### 八、爱因斯坦是如何定义"科学（Science）"的？

爱因斯坦的定义：

> **"西方科学的形成是以两个伟大的成就为基础的，一是古希腊哲学家们发明的记录在欧几里德几何学中的形式逻辑体系；一是文艺复兴时期发现进行系统的实验有可能找到自然现象之间的因果关系。"**

爱因斯坦还说过："我们推崇古代希腊是西方科学的摇篮。在那里，世界第一次目睹了一个逻辑体系的奇迹，这个逻辑体系如此精密地一步一步地推进，以致它的每一个命题都是绝对不容置疑的——我这里说的就是欧几里德几何。"

### 九、爱因斯坦对我国先贤在这方面有何点评？

爱因斯坦在回应"李约瑟问题"时指出：

> **"中国的先贤们没有走出这两步，在我看来不值得大惊小怪。这些发现居然被人类做了出来才令人吃惊。"**

（原文：_"In my opinion one need not be astonished that the Chinese sages did not make these steps. The astonishing thing is that these discoveries were made at all."_）

"这两步"即指爱因斯坦定义的科学的两大支柱：形式逻辑体系和系统的实验方法。爱因斯坦认为中国先贤没有发展出这两样东西并不可怪，真正令人惊叹的是人类居然能创造出这些发现。

课程文件2.7中也提到爱因斯坦的感叹："他们是怎么想出来的？我们中国的古人与现在的中国人一样，在智力上决不弱于他人。但是，要传承古人，后人是需要实力说话的。"

### 十、爱因斯坦认为中国古代有科学吗？

爱因斯坦认为中国古代**没有符合"Science"本意的科学**。

根据课程文件10.2的阐述：爱因斯坦强调的现代科学的两大支柱——形式逻辑体系和系统的实验方法——都起源于古希腊和文艺复兴时期的欧洲。中国的先贤没有走出这两步。课程明确指出："历史上没有符合《科学启蒙》课程所介绍的符合英文'Science'本意的'中国科学'！"中国古代有的是技术和文明（technology），而非科学（science）。正如课程引用培根所言，中国古代"四大发明"是"机械发明"（mechanical discoveries），属于技术范畴。

**出处：** [10.2 "西学中源"实乃误国之妄言.txt](test_articles/10.2 "西学中源"实乃误国之妄言.txt)
