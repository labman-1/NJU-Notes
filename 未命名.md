### 一、 第3、4题严谨步骤解答

为了方便你对照检查，这里给出第3题和第4题勾选方程的完整、严谨求解步骤。在求解全微分方程时，除了常用的凑微分法，我们在本节中采用更为规范和严密的**“偏积分法”**进行推导。

---

#### 第3题：确定常数 $A$，使微分方程成为全微分方程并求解

##### (1) $(x^2 + 3xy)dx + (Ax^2 + 4y)dy = 0$

*   **第一步：确定常数 $A$**
    设 $P(x,y) = x^2 + 3xy$， $Q(x,y) = Ax^2 + 4y$。
    要使方程为全微分方程，必须满足交叉偏导数相等条件：
    $$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$$
    分别求偏导：
    $$\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(x^2 + 3xy) = 3x$$
    $$\frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(Ax^2 + 4y) = 2Ax$$
    令两式相等：
    $$3x = 2Ax \implies 2A = 3 \implies A = \frac{3}{2}$$

*   **第二步：求解微分方程**
    将 $A = \frac{3}{2}$ 代回原方程，得：
    $$\left(x^2 + 3xy\right)dx + \left(\frac{3}{2}x^2 + 4y\right)dy = 0$$
    设该全微分对应的函数为 $u(x,y)$，则有 $\frac{\partial u}{\partial x} = x^2 + 3xy$。
    对 $x$ 进行偏积分：
    $$u(x,y) = \int (x^2 + 3xy) dx = \frac{1}{3}x^3 + \frac{3}{2}x^2y + \varphi(y)$$
    （其中 $\varphi(y)$ 是关于 $y$ 的待定函数）
    
    再将 $u(x,y)$ 对 $y$ 求偏导，并令其等于 $Q(x,y)$：
    $$\frac{\partial u}{\partial y} = \frac{3}{2}x^2 + \varphi'(y) = \frac{3}{2}x^2 + 4y$$
    从而：
    $$\varphi'(y) = 4y \implies \varphi(y) = 2y^2$$
    故原函数为：
    $$u(x,y) = \frac{1}{3}x^3 + \frac{3}{2}x^2y + 2y^2$$
    
    **结论：**
    常数 $A = \frac{3}{2}$，方程通解为：
    $$\frac{1}{3}x^3 + \frac{3}{2}x^2y + 2y^2 = C \quad （或写为\ 2x^3 + 9x^2y + 12y^2 = C_1）$$

---

##### (4) $\left(\frac{1}{x^2} + \frac{1}{y^2}\right)dx + \frac{Ax+1}{y^3}dy = 0$  （定义域：$x \neq 0, y \neq 0$）

*   **第一步：确定常数 $A$**
    设 $P(x,y) = x^{-2} + y^{-2}$， $Q(x,y) = (Ax+1)y^{-3} = Axy^{-3} + y^{-3}$。
    根据全微分判定条件：
    $$\frac{\partial P}{\partial y} = -2y^{-3}$$
    $$\frac{\partial Q}{\partial x} = Ay^{-3}$$
    令二者相等，得：
    $$-2y^{-3} = Ay^{-3} \implies A = -2$$

*   **第二步：求解微分方程**
    将 $A = -2$ 代回原方程，得：
    $$\left(\frac{1}{x^2} + \frac{1}{y^2}\right)dx + \frac{-2x+1}{y^3}dy = 0$$
    设其原函数为 $u(x,y)$，则有 $\frac{\partial u}{\partial x} = x^{-2} + y^{-2}$。
    对 $x$ 偏积分：
    $$u(x,y) = \int (x^{-2} + y^{-2}) dx = -x^{-1} + xy^{-2} + \varphi(y) = -\frac{1}{x} + \frac{x}{y^2} + \varphi(y)$$
    对 $y$ 求偏导，令其等于 $Q(x,y)$：
    $$\frac{\partial u}{\partial y} = -2xy^{-3} + \varphi'(y) = \frac{-2x}{y^3} + \varphi'(y)$$
    将其与 $Q(x,y) = \frac{-2x+1}{y^3} = \frac{-2x}{y^3} + \frac{1}{y^3}$ 对比：
    $$\varphi'(y) = y^{-3} \implies \varphi(y) = -\frac{1}{2}y^{-2} = -\frac{1}{2y^2}$$
    故原函数为：
    $$u(x,y) = -\frac{1}{x} + \frac{x}{y^2} - \frac{1}{2y^2}$$

    **结论：**
    常数 $A = -2$，方程通解为：
    $$\frac{x}{y^2} - \frac{1}{x} - \frac{1}{2y^2} = C$$

#### 第4题：确定函数并求解

##### (1) $P(x,y)dx + (2ye^x + y^2e^{3x})dy = 0$

*   **第一步：求待定函数 $P(x,y)$**
    设 $Q(x,y) = 2ye^x + y^2e^{3x}$。
    全微分方程要求 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$。
    先求 $Q$ 对 $x$ 的偏导：
    $$\frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(2ye^x + y^2e^{3x}) = 2ye^x + 3y^2e^{3x}$$
    因此：
    $$\frac{\partial P}{\partial y} = 2ye^x + 3y^2e^{3x}$$
    两边对 $y$ 偏积分，确定 $P(x,y)$ 的形式：
    $$P(x,y) = \int (2ye^x + 3y^2e^{3x}) dy = y^2e^x + y^3e^{3x} + \psi(x)$$
    （其中 $\psi(x)$ 是关于 $x$ 的任意一阶可导函数。最简情况下可取 $\psi(x) = 0$，此处我们保留其最严谨的通式形式）

*   **第二步：求解微分方程**
    设原函数为 $u(x,y)$，则有 $\frac{\partial u}{\partial y} = Q(x,y) = 2ye^x + y^2e^{3x}$。
    对 $y$ 偏积分：
    $$u(x,y) = \int (2ye^x + y^2e^{3x}) dy = y^2e^x + \frac{1}{3}y^3e^{3x} + \eta(x)$$
    对 $x$ 求偏导：
    $$\frac{\partial u}{\partial x} = y^2e^x + y^3e^{3x} + \eta'(x)$$
    令其等于 $P(x,y) = y^2e^x + y^3e^{3x} + \psi(x)$，得：
    $$\eta'(x) = \psi(x) \implies \eta(x) = \int \psi(x) dx$$

    **结论：**
    待定函数为：
    $$P(x,y) = y^2e^x + y^3e^{3x} + \psi(x) \quad （\psi(x)\text{为任意一阶可导函数}）$$
    微分方程的通解为：
    $$y^2e^x + \frac{1}{3}y^3e^{3x} + \int \psi(x) dx = C$$
    *(注：若取最简情况 $\psi(x)=0$，则 $P = y^2e^x + y^3e^{3x}$，通解为 $y^2e^x + \frac{1}{3}y^3e^{3x} = C$)*

---

##### (2) $(x^{-2}y^{-2} + xy^{-3})dx + Q(x,y)dy = 0$

*   **第一步：求待定函数 $Q(x,y)$**
    设 $P(x,y) = x^{-2}y^{-2} + xy^{-3}$。
    根据全微分条件要求 $\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$。
    先求 $P$ 对 $y$ 的偏导：
    $$\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(x^{-2}y^{-2} + xy^{-3}) = -2x^{-2}y^{-3} - 3xy^{-4}$$
    因此：
    $$\frac{\partial Q}{\partial x} = -2x^{-2}y^{-3} - 3xy^{-4}$$
    对 $x$ 偏积分，确定 $Q(x,y)$：
    $$Q(x,y) = \int (-2x^{-2}y^{-3} - 3xy^{-4}) dx = 2x^{-1}y^{-3} - \frac{3}{2}x^2y^{-4} + \phi(y) = \frac{2}{xy^3} - \frac{3x^2}{2y^4} + \phi(y)$$
    （其中 $\phi(y)$ 是关于 $y$ 的任意一阶可导函数）

*   **第二步：求解微分方程**
    设原函数为 $u(x,y)$，由 $\frac{\partial u}{\partial x} = P(x,y) = x^{-2}y^{-2} + xy^{-3}$，对 $x$ 偏积分：
    $$u(x,y) = \int (x^{-2}y^{-2} + xy^{-3}) dx = -x^{-1}y^{-2} + \frac{1}{2}x^2y^{-3} + \theta(y) = -\frac{1}{xy^2} + \frac{x^2}{2y^3} + \theta(y)$$
    对 $y$ 偏导：
    $$\frac{\partial u}{\partial y} = 2x^{-1}y^{-3} - \frac{3}{2}x^2y^{-4} + \theta'(y)$$
    对照 $Q(x,y)$，易知 $\theta'(y) = \phi(y)$。

    **结论：**
    待定函数为：
    $$Q(x,y) = \frac{2}{xy^3} - \frac{3x^2}{2y^4} + \phi(y) \quad （\phi(y)\text{为任意一阶可导函数}）$$
    微分方程的通解为：
    $$-\frac{1}{xy^2} + \frac{x^2}{2y^3} + \int \phi(y) dy = C$$
    *(注：若取最简情况 $\phi(y)=0$，则 $Q = \frac{2}{xy^3} - \frac{3x^2}{2y^4}$，通解为 $-\frac{1}{xy^2} + \frac{x^2}{2y^3} = C$)*

### 二、 积分因子（微分因子）的数学推导与解题策略

既然你对这部分很感兴趣，我们来从底层理清**“积分因子公式是怎么写出来的”**。

对于不恰当的方程 $Pdx + Qdy = 0$，其偏导数不相等（$\frac{\partial P}{\partial y} \neq \frac{\partial Q}{\partial x}$）。我们引入一个非零乘子 $\mu(x,y)$，使新方程：
$$\mu P dx + \mu Q dy = 0$$
成为全微分方程。

#### 1. 寻找 $\mu(x,y)$ 的偏微分方程
为了使新方程成为全微分方程，它必须通过“交叉偏导数检验”：
$$\frac{\partial (\mu P)}{\partial y} = \frac{\partial (\mu Q)}{\partial x}$$
根据偏导数乘积法则展开：
$$\mu \frac{\partial P}{\partial y} + P \frac{\partial \mu}{\partial y} = \mu \frac{\partial Q}{\partial x} + Q \frac{\partial \mu}{\partial x}$$
移项整理得：
$$Q \frac{\partial \mu}{\partial x} - P \frac{\partial \mu}{\partial y} = \mu \left( \frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x} \right)$$
这是一个关于未知函数 $\mu(x,y)$ 的一阶偏微分方程，通常很难直接求解。为了打破僵局，数学家们考虑了两种最经典的简化情况：**单变量积分因子**。

---

#### 2. 情况 A：假设积分因子仅与 $x$ 有关，即 $\mu = \mu(x)$
由于 $\mu$ 只是 $x$ 的函数，它对 $y$ 的偏导数为零：
$$\frac{\partial \mu}{\partial y} = 0 \quad \text{且} \quad \frac{\partial \mu}{\partial x} = \frac{d\mu}{dx}$$
将这两个条件代回刚才的偏微分方程中：
$$Q \frac{d\mu}{dx} = \mu \left( \frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x} \right)$$
分离变量（把所有包含 $\mu$ 的项挪到左边，其余挪到右边）：
$$\frac{1}{\mu} \frac{d\mu}{dx} = \frac{\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}}{Q}$$

*   **判别条件：** 只有当右边的表达式 $\frac{\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}}{Q}$ **仅仅包含 $x$（或者为常数）**时，这种假设才成立。我们设该函数为 $F(x)$。
*   **求解 $\mu(x)$：**
    两边对 $x$ 积分：
    $$\ln |\mu| = \int F(x) dx \implies \mu(x) = e^{\int F(x) dx}$$

---

#### 3. 情况 B：假设积分因子仅与 $y$ 有关，即 $\mu = \mu(y)$
同理，此时 $\frac{\partial \mu}{\partial x} = 0$，且 $\frac{\partial \mu}{\partial y} = \frac{d\mu}{dy}$。代回方程可得：
$$-P \frac{d\mu}{dy} = \mu \left( \frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x} \right)$$
分离变量：
$$\frac{1}{\mu} \frac{d\mu}{dy} = \frac{\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}}{P}$$

*   **判别条件：** 只有当右边的表达式 $\frac{\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}}{P}$ **仅仅包含 $y$（或者为常数）**时，此假设成立。我们设该函数为 $G(y)$。
*   **求解 $\mu(y)$：**
    两边对 $y$ 积分：
    $$\mu(y) = e^{\int G(y) dy}$$

---

### 三、 实战检验提示

掌握了这两个公式后，在面对非全微分方程时，我们的解题策略如下：
1.  计算偏导差值：$\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}$。
2.  **先试除以 $Q$**：看 $\frac{\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}}{Q}$ 是不是纯 $x$ 的函数。如果是，直接套用 $\mu(x) = e^{\int F(x)dx}$；
3.  **再试除以 $-P$**（或算 $\frac{\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}}{P}$）：看是不是纯 $y$ 的函数。如果是，直接套用 $\mu(y) = e^{\int G(y)dy}$；
4.  乘以求得的 $\mu$ 后，原方程转化为全微分方程，再用前面的方法求解。
